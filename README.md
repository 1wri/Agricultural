# Agricultural
扶贫助农管理系统 助农产品销售管理系统 基于springboot+vue的前后端分离农产品销售管理系统


语言: Java 数据库：MySQL  前后端分离

前端技术 : Vue + ElementUl

后端技术 : SpringBoot + MyBatisPlus

系统角色：管理员、店家、用户

管理员：用户\店家管理、店铺\商品管理、商品分类管理、生产地分类管理、爱心捐赠管理、贫困店家申请管理、贫困店家信息管理、扶贫政策管理、爱心箱管理、退货申请管理、系统\订单管理

店家：注册登录、店铺\商品管理、商品分类管理、生产地分类管理、贫困店家申请管理、订单管理

用户：注册登录、首页、店铺、商品、扶贫政策、爱心箱、自然风光介绍、个人中心、购物车、订单管理、收藏
